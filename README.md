# Quran Focus 🕌

A beautiful, minimalist web application for focused Quran reading and listening. Built with pure HTML, CSS, and JavaScript, this project is vibe-coded to create a serene and immersive experience for Quranic contemplation.

## 🎯 Vision

Quran Focus is guided by two principles:

1. **Focus on the Quran, not the app.** The interface should fade into the background so the reader's attention stays on the words of Allah. Features that add visual clutter, draw the eye away from the ayah, or make the app feel "busy" go against this vision — even if they're technically useful. Minimalism here isn't an aesthetic choice; it's a commitment to purpose.

2. **Authenticated, approved sources only.** All content — audio and text — comes from sources trusted by scholars and students of knowledge. The default reciter is used by students of knowledge, and the default translation is Muhsin Khan & Hilali, which has been approved by scholars. This project does not use unverified or questionable sources.

## 🌟 Features

- **Minimalist Design**: Clean, distraction-free interface for focused reading
- **Ambient Particles**: Beautiful particle animations in the background
- **Audio Recitation**: High-quality audio playback of Quranic verses
- **Progressive Web App (PWA)**: Installable on any device for offline access
- **Responsive Design**: Works seamlessly across all devices
- **Fullscreen Mode**: Immersive viewing experience
- **Surah & Ayah Selection**: Easy navigation through the entire Quran
- **Beautiful Typography**: Optimized Arabic and English fonts
- **Onboarding Guide**: A quick introduction for first-time users.

## 🛠️ Technical Details

- Built with vanilla JavaScript, HTML5, and CSS3
- Uses Particles.js for interactive background effects
- Optimized font loading for Arabic and English text
- Service Worker implementation for offline functionality
- Responsive design with mobile-first approach
